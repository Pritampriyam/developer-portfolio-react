# 🚀 Pritam Kumar – Developer Portfolio

A modern, responsive **full-stack developer portfolio** built using React and Tailwind CSS. This portfolio showcases projects, experience, skills, and provides a contact system powered by EmailJS.

---

## 📌 Features

### 🎨 UI & UX

* Fully responsive modern design
* Glassmorphism + animated UI
* Smooth scroll navigation
* Interactive sections (Hero, About, Projects, etc.)

### ⚡ Functional Features

* 📬 Contact form with EmailJS integration
* 📱 Mobile-friendly navigation menu
* 🎯 Dynamic project showcase
* 🧭 Smooth section-based routing
* 💡 Animated components and transitions

## ⚙️ Environment Variables

To enable the contact form, create a `.env` file:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
